POLÍTICA DE PRIVACIDADE

Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações pessoais.

Última atualização: Janeiro 2025

IMPORTANTE

A Robux Premium leva a privacidade dos seus dados muito a sério. Esta política foi elaborada em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

1. DEFINIÇÕES

Para os efeitos desta Política de Privacidade:

  • Dados Pessoais: Qualquer informação relacionada a pessoa natural identificada ou identificável.
  • Titular: Pessoa natural a quem se referem os dados pessoais que são objeto de tratamento.
  • Tratamento: Toda operação realizada com dados pessoais.
  • Controlador: Pessoa natural ou jurídica a quem competem as decisões referentes ao tratamento de dados.

2. DADOS QUE COLETAMOS

Coletamos apenas os dados estritamente necessários para a prestação dos nossos serviços:

Tipo de Dado Finalidade Base Legal
Endereço de e-mail Entrega do código PIN e comunicação Execução de contrato
Informações de transação Processamento de pagamentos via PIX Obrigação legal
Endereço IP Segurança e prevenção de fraudes Legítimo interesse

3. COMO USAMOS SEUS DADOS

Utilizamos seus dados pessoais para as seguintes finalidades:

  1. Processamento de pedidos: Para entregar o código PIN de Robux adquirido.
  2. Comunicação: Para enviar confirmações de pedido e atualizações.
  3. Suporte ao cliente: Para responder suas dúvidas e solicitações.
  4. Segurança: Para prevenir fraudes e atividades suspeitas.
  5. Melhorias: Para aprimorar nossos serviços e experiência do usuário.

4. COMPARTILHAMENTO DE DADOS

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto nas seguintes situações:

  1. Processadores de pagamento: Para processar transações via PIX.
  2. Requisito legal: Quando exigido por lei, regulamento ou processo legal.
  3. Proteção de direitos: Para proteger nossos direitos, privacidade, segurança ou propriedade.

5. RETENÇÃO DE DADOS

Mantemos seus dados pessoais apenas pelo tempo necessário para cumprir as finalidades para as quais foram coletados, incluindo para fins de cumprimento de obrigações legais, contábeis ou de relatórios.

  • Dados de transação: 5 anos (conforme legislação fiscal)
  • Dados de conta: Enquanto sua conta estiver ativa
  • Registros de segurança: 1 ano

6. SEUS DIREITOS (LGPD)

Conforme a Lei Geral de Proteção de Dados, você tem os seguintes direitos:

Confirmação e Acesso

Confirmar a existência de tratamento e acessar seus dados.

Correção

Solicitar correção de dados incompletos, inexatos ou desatualizados.

Eliminação

Solicitar a eliminação de dados tratados com seu consentimento.

Revogação

Revogar o consentimento a qualquer momento.

7. SEGURANÇA DOS DADOS

Implementamos medidas de segurança técnicas e organizacionais apropriadas para proteger seus dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição.

  • Criptografia SSL/TLS para todas as transações
  • Armazenamento seguro em servidores com proteção avançada
  • Acesso restrito a dados pessoais apenas para funcionários autorizados
  • Monitoramento contínuo de sistemas para detecção de atividades suspeitas

8. COOKIES E TECNOLOGIAS SIMILARES

Utilizamos cookies essenciais para o funcionamento do site. Não utilizamos cookies de rastreamento ou para fins publicitários.

9. MENORES DE IDADE

Nossos serviços não são direcionados a menores de 18 anos. Se tomarmos conhecimento de que coletamos dados pessoais de menores sem verificação do consentimento parental, tomaremos medidas para remover essas informações.

10. ALTERAÇÕES NESTA POLÍTICA

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas publicando a nova política nesta página com data de atualização revisada.
